# Avito ↔ Telegram Bridge (clean)

Чистая рабочая интеграция:
- Автоответы бота (LLM + FAQ + статконтекст).
- Если бот пишет фразу вроде "По данному вопросу вам в ближайшее время ответит наш менеджер." — включается бесконечная пауза до ответа менеджера.
- После ответа менеджера — 15-минутный cooldown, затем бот снова активен.
